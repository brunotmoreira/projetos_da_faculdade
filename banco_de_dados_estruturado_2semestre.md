o projeto do segundo semestre foi voltado para uma empresa varejista de moda que atua através de eccomerce. 
A principal dor identificada foi a falta de organição e visualização dos produtos para comercialização, resultando em vendas de produtos que já não estavam mais disponíveis em estoque. 

Cliente disponibilou algumas bases excel onde registrava as vendas, gastos e produtos que comprava dos fornecedores, contudo não relacionavam as informações para identificar quanto dos produtos de determinado tipo ainda estavam em estoque. 
Abaixo vou compartilhar o modelo relacional para facilitar para a criação do banco de dados.
