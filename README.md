# projetos_da_faculdade
Repositorio para armanezar os projetos interdiciplinares confecionados semestralmente na faculdade.
Os projetos são realizados em conjunto com pequenas e médias empresas reais, onde o aluno identifica as necessidades do cliente e oferece uma solução para o problema identificado. 
Nomes e dados relacionados as empresas serão omitidos.
